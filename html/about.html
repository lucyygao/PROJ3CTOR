ABOUT

PROJ3CTOR is centered around modeling and sculpting a 3D mesh from 2D user input. Using implicit surfaces and point rendering, it generates organic volumes in a 3D space based on the extruded projections of mouse input from two perpendicularly positioned 2D screens.

The top 2D screen is used to sketch the basic profile of the shape, whether it be a circle, a square, or a lengthy squiggle. Any shape the user draws on the first screen will be extruded and shown on the second screen with dashed lines to hint at its position, from which the user can further edit. The second screen is positioned perpendicular to the first screen, and the user can then use this screen to refine the 3D model, carving and sculpting out the true shape they want.

See the documentation page for more instructions or the demo page for some inspiration.

This project was created by Lucy Gao under the mentorship of Santiago Lombeyda and Professor George Djorgovski for SURF 2021.
