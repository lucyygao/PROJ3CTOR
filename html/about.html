ABOUT

PROJ3CTOR is centered around modeling and sculpting of a 3D mesh from 2D user input. It generates organic volumes in 3D based on the extruded projections of mouse input from two perpendicularly positioned 2D screens using implicit surfaces and point rendering.

The top screen is used for sketching the basic profile of the mesh-- for example, a circle for a sphere or cylinder, or a triangle for a triangular prism or a triangular pyramid. The second screen is positioned perpendicular to the first screen, and the user can use this screen to refine the 3D model, carving and sculpting out the true shape they want. Any shape the user draws on the first screen will be extruded and shown on the second screen with dashed lines to hint at its position, from which the user can further edit.

This project was created by Lucy Gao under the mentorship of Santiago Lombeyda and Professor George Djorgovski for SURF 2021.
